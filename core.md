# Usage
## Installation

Core Nixer plugin is distributed through [Maven Central](https://search.maven.org/search?q=io.nixer).

It requires dependency to Core Nixer plugin as well.

```kotlin
dependencies {
    implementation("io.nixer:nixer-plugin-core:0.1.0.0")
}
```
